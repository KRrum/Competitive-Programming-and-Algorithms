# Competitive Programming & Algorithms Library 🚀

Welcome! This repository serves as a personal archive of my algorithmic journey, featuring solutions to complex problems and highly optimized implementations of classic data structures and algorithms.

> **Note for Recruiters:** The code here reflects my focus on computational complexity (Big O), memory efficiency, and maintaining clean, documented code in a high-performance context.

---

## 📂 Repository Structure

The project is organized by competition and topic to ensure easy navigation:

* **`/oi` (Polish Olympiad in Informatics):** Solutions to problems from the prestigious **Olimpiada Informatyczna**. 
    * Folders are organized by edition (e.g., `/31` for the XXXI edition).
    * *Status:* Solutions marked as `100/100` have passed all official test cases on the Szkopuł platform.
* **`/Sorting`:** A collection of sorting algorithms. I focus on low-level optimizations, such as bitwise operations in Python or tail-recursion handling.
* **`/Data-Structures`:** (In progress) Custom implementations of Segment Trees, Graphs, and Priority Queues.

## 🛠️ Engineering Standards
For every significant problem or algorithm, I aim to provide:
1.  **Optimized Source Code** (C++ or Python) with comments explaining the "why" behind the logic.
2.  **Local README** (within the task folder) containing a brief strategy analysis and a link to the problem statement.
3.  **Complexity Analysis** for both time and space.
